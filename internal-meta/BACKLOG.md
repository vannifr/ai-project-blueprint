## **VP‑02 (CRITICAL) – Module 01.07 “Bewijsstandaarden” daadwerkelijk invullen \[x\]**

- **Probleem:** er wordt verwezen naar “strikte standaarden”, maar de standaarden ontbreken → Gate reviews zijn niet uitvoerbaar.
- **Aanpassing (concreet):**
    1. [x] Maak **Module 01.07: Bewijsstandaarden** met een **normentabel** per risicoklasse (*Minimaal / Beperkt / Hoog*).
    1. [x] Definieer per risicoklasse minimaal:
        - **Welke tests verplicht zijn** (inhoud, veiligheid, eerlijkheid, stabiliteit).
        - **Welke drempels gelden** (of: drempels moeten ingevuld worden + verplicht akkoord Guardian).
        - **Welke loggegevens verplicht zijn** (wat leggen we vast, hoe lang bewaren).
    1. [x] Koppel deze standaarden expliciet aan elke Gate: “Gate X vereist Bewijs A, B, C”.
- **Waar:** nieuwe Module 01.07 + verwijzingen in Modules 03/04/06/07 + update Gate‑checklists.
- **Output:** Module 01.07 + geüpdatete Gate‑checklists (met “bewijs vereist”).
- **Klaar wanneer:** elke Gate checklist verwijst naar **concrete** eisen (geen vage termen zoals “volledige validatie”).
