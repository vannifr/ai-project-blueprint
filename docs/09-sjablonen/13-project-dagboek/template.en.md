# Project Diary — Template

Weekly log for AI projects. Use this diary to track progress, decisions, and lessons learned. Keep it brief and factual — this is a memory aid, not a status report.

!!! tip "Frequency"
    Fill in at minimum **weekly**, preferably on Friday. Also complete the gate section after every Gate Review.
