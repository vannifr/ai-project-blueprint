# Project Dagboek — Template

Wekelijks logboek voor AI-projecten. Gebruik dit dagboek om voortgang, beslissingen en leerpunten bij te houden. Houd het kort en feitelijk — dit is geen statusrapport maar een geheugensteun.

!!! tip "Invulfrequentie"
    Vul minimaal **wekelijks** in, bij voorkeur op vrijdag. Noteer bij elke Gate Review ook de gate-sectie.
