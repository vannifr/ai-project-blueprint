# 1. What are Core Principles?

## 1. When Do We Apply the Core Principles?

A project qualifies for this approach when the following three criteria are met:

| Criterion              | Requirement                                                                                 |
| :--------------------- | :------------------------------------------------------------------------------------------ |
| **Material Impact**    | The system relies on AI for production outputs or decisions that affect the business.       |
| **Explicit Context**   | Inputs that steer behaviour (**Prompts**, RAG) are managed as versioned artefacts.          |
| **Continuous Testing** | Changes undergo validation specifically designed for the non-deterministic behaviour of AI. |

> **Once qualified**, the operational controls for governance, monitoring and traceability apply to guide development.
